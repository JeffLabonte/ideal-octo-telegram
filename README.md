# ideal-octo-telegram

API to control your devices on your network and make all your automation scripts available at one place


## Requirements:

* Docker
* Python 3.9
* Postgresql
* [Poetry](https://python-poetry.org/docs/#installation)


## Getting Started:

* Run `make install`
